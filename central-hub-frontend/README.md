# Central Hub Frontend

Vanilla JavaScript + Tailwind CDN ile hazırlanmış statik frontend.

## Çalıştırma

```bash
python3 -m http.server 5500
```

Sonra:

```text
http://localhost:5500
```

## Klasör Yapısı

```text
central-hub-frontend/
├── index.html
├── assets/
│   ├── css/style.css
│   ├── js/api.js
│   ├── js/main.js
│   └── images/logo.svg
└── modules/
    ├── club-management/index.html
    ├── academic-planner/index.html
    └── security-cameras/index.html
```
